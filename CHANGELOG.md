# Changelog

## v1.0.0 - 2026-07-01

First tagged portfolio baseline for the SaaS Backend Starter project.

### Highlights

- Complete Spring Boot 3.5 backend starter built on Java 21.
- JWT authentication with refresh tokens.
- Role-based and permission-based access control.
- Multi-tenant user, organization, role, permission, file, and audit-log domains.
- Swagger/OpenAPI documentation.
- Local Docker Compose demo setup with MariaDB.
- Unit and web-layer tests for key controllers, services, and exception handling.

### Quality

- Added GitHub Actions CI for Maven test and package verification.
- Documented CI status in the project README.

### Scope

- This release is a portfolio/demo baseline.
- Production deployment hardening, external secret management, database migrations, and infrastructure automation are intentionally outside this release.
